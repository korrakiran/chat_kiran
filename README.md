# WordTrack

![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)

A lightweight, open-source tool for writers to track daily word counts and maintain creative momentum.

## Features

- 📊 **Real-time word count tracking** for multiple projects
- 🎯 **Daily goal setting** with progress visualization
- 📅 **Historical data** export (CSV/JSON)
- 🖥️ **Cross-platform compatibility** (Windows, macOS, Linux)
- 🔒 **Local data storage** (no cloud dependency)

## Installation

### Node.js
```bash
npm install -g wordtrack
wordtrack init
```

### Python
```bash
pip install wordtrack
wordtrack --setup
```

### Desktop App
Download pre-built binaries from [releases](https://github.com/yourusername/wordtrack/releases)

## Usage

1. **Start tracking**:
```bash
wordtrack start "novel_draft.txt"
```

2. **Set daily goal**:
```bash
wordtrack set-goal 1500
```

3. **Check progress**:
```bash
wordtrack status
```

4. **Export data**:
```bash
wordtrack export --format csv --days 30
```
